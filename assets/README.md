# Shotchi Assets

This folder contains all visual assets for the Shotchi MVP.

## Character States (Adi the Blob)

Location: `assets/character/`

| File | State | Usage |
|------|-------|-------|
| `adi-happy.png` | Happy | Shown when injection logged on time |
| `adi-neutral.png` | Neutral | Shown when injection due today, not yet logged |
| `adi-sad.png` | Sad | Shown when injection is overdue |

**Replacement notes:**
- Keep same filenames for easy swap
- Recommended size: 512x512px minimum
- Format: PNG with transparency preferred
- Style: Clean vector, minimal, consistent across all 3 states

## App Icons

Location: `assets/icons/`

| File | Usage |
|------|-------|
| `app-icon.png` | Main app icon (iOS/Android) |

**Replacement notes:**
- Recommended size: 1024x1024px for stores
- Will be resized for various icon requirements
- Format: PNG, square, no transparency (solid background)
